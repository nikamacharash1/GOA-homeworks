ECMAScript-ის შესახებ

რა არის ECMAScript?

ECMAScript (მოკლედ: ES) არის სკრიპტების ენების სტანდარტი, რომელიც განსაზღვრავს სინტაქსს და ფუნქციონალობას. ეს სტანდარტი გამოიყენება ისეთი ენებში, როგორებიცაა JavaScript, JScript და ActionScript.

მთავარი მიზანი

ECMAScript-ის მიზანია სკრიპტინგ ენების სტანდარტიზაცია, რათა მათ შორის თავსებადობა და ერთგვაროვნება შეინარჩუნოს. ეს ეხმარება პროგრამისტებს, რომ ერთ სტანდარტზე იმუშაონ სხვადასხვა პლატფორმებზე.

ტემპორალური ისტორია

  • 1997 წელი: ECMA-262 სტანდარტის პირველი ვერსია.
  • 2009 წელი: ECMAScript 5 (ES5) — ახალი შესაძლებლობები, როგორიცაა "strict mode".
  • 2015 წელი: ECMAScript 6 (ES6) — ახალი შესაძლებლობები, როგორიცაა arrow functions, classes, promises და modules.

მნიშვნელოვანი ვერსიები

ECMAScript-ის რამდენიმე მნიშვნელოვანი ვერსია მოიცავს:

  • ES3 (1999): პირველი დიდი ვერსია.
  • ES5 (2009): JSON მხარდაჭერა, "strict mode".
  • ES6 (2015): კლასები, arrow functions, promises და მოდულები.

მნიშვნელოვანი თვისებები

  • Arrow functions: უფრო კომპაქტური და გამარტივებული ფუნქციები.
  • let/const: გაუმჯობესებული ცვლადების მართვა.
  • Promises: ასინქრონული ოპერაციების უმარტივესი მართვა.
  • Template Literals: დინამიკური სტრიქონები `${}`-თან.
